## JSON Configuration

```
{
    "API_URL": "http://10.205.0.2:3011",
    "IS_MOVIE": false,
    "TYPE_READER": "8h10d"
}
```
----

### IS_MOVIE options:
If false, do not show the video on the first screen
- true
- false (default)

----

### TYPE_READER options:
RFID card reader conversion
- 8h10d (default) : base 10
- 10h13d : base 13
- 10h : hexadecimal
